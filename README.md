## Overview

NLP toolkit and model hub supporting all 11 official South African languages. Provides translation, sentiment analysis, speech-to-text, and text-to-speech capabilities with offline-first architecture.

## Features

- **Multi-language NLP** — Support for Zulu, Xhosa, Afrikaans, English, Sesotho, Tswana, Tsonga, Swati, Venda, Ndebele, Pedi
- **Translation Models** — Neural machine translation between SA languages
- **Sentiment Analysis** — Culturally-aware sentiment detection
- **Speech Processing** — STT and TTS for low-resource languages
- **Offline-First** — Runs without internet using optimized local models

## Quick Start

```bash
git clone https://github.com/Raphasha27/SA-Language-AI.git
cd SA-Language-AI
pip install -r requirements.txt
python run.py
```
